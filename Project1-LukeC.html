Intro and Goal

We will be looking at the College Football Transfer Portal, it is current as of 2023.

Every year, more and more players enter the transfer portal in hopes of obtaining a starting position, earning more money, etc.

My goal is to see if we can build a model that can predict whether a a player will transfer to a Powerhouse program or not, as well as determine what factors strongly contribute to a transfer decision.

Target Variable - PowerHouse Schools

Now that we know what schools have the highest rating, we can create a target variable.

model_data <- data %>%
  filter(!is.na(stars) & !is.na(destination))

power_schools <- c("Alabama", "Georgia", "Ohio State", "Michigan", "Texas", "USC", "LSU", "Oklahoma", "Oregon", "Notre Dame",
"Ole Miss", "Auburn", "Florida", "Tennessee", 
"Penn State", "Texas A&M")

Building Model

We adjust the variables so that they can fit the logistic regression model by converting values and formatting.

model_data <- model_data %>%
  mutate(
    Is_Powerhouse = ifelse(destination %in% power_schools, 1, 0),
    Origin_Type = ifelse(origin %in% power_schools, "From Power 4", "From Mid-Major"),
    Position = ifelse(position %in% c("QB","WR","RB","TE","OL","DL","LB","DB","CB","S"), 
                      position, "Other")
  )
scoreModel <- glm(Is_Powerhouse ~ stars + Origin_Type + position, 
                  data = model_data, 
                  family = "binomial")
impact_scores <- as.data.frame(summary(scoreModel)$coefficients)
impact_scores$Variable <- rownames(impact_scores)
colnames(impact_scores)[1] <- "Impact"

Setting Impact Scores

impact_scores <- impact_scores %>% 
  filter(Variable != "(Intercept)") %>%
  mutate(Variable = gsub("position", "", Variable)) %>%
  mutate(Variable = gsub("Origin_Type", "", Variable)) %>%
  arrange(Impact)

Strongest Contributing Factors

The strongest contributing factors are stars and certain positions on the field. Punter being the greatest, due to the scarcity at the position.

ggplot(impact_scores, aes(x = reorder(Variable, Impact), y = Impact, fill = Impact > 0)) + geom_col() + coord_flip() +
  scale_fill_manual(values = c("firebrick", "forestgreen"), labels = c("Hurts Chances", "Helps Chances")) +
  geom_hline(yintercept = 0, color = "black") +
  labs(title = "What Drives a Transfer to a Powerhouse?", x = NULL, y = "Impact Score")

Predicting Transfer Decision

new_player <- data.frame(
  stars = 3,                      
  Origin_Type = "From Mid-Major",   
  eligibility = "Immediate",
  position = "CB"
)
prob <- predict(scoreModel, new_player, type = "response")
print(paste("Player Profile:", new_player$stars, "Star", new_player$position,",", new_player$Origin_Type))
## [1] "Player Profile: 3 Star CB , From Mid-Major"
print(paste("Probability of Powerhouse Destination:", round(prob * 100, 1), "%"))
## [1] "Probability of Powerhouse Destination: 4.9 %"

We see that a 3-star CB Recruit from a Mid-Major conference has a very low percentage of transferring to a PowerHouse School.

Model Accuracy

Lastly we find the model accuracy:

probs <- predict(scoreModel, type = "response")
predictions <- ifelse(probs > 0.5, 1, 0)
accuracy <- mean(predictions == model_data$Is_Powerhouse)
table(Predicted = predictions, Actual = model_data$Is_Powerhouse)
##          Actual
## Predicted    0    1
##         0 5608  391
##         1   13    7
print(paste("Total Model Accuracy:", round(accuracy * 100, 1), "%"))
## [1] "Total Model Accuracy: 93.3 %"

93% accuracy is not a bad start. This model is pretty reliable, and the more data we add, the better it will get.

Conclusion

We found that the strongest factors that influence a transfer decision were:

-Position
-Star Rating

Using those factors, we were able to predict the probability of a player in the portal transferring to a Power 4 Conference or not. All with a 93% accuracy score! This model could help coaching staff know who to target based on what conference they’re in and where the player’s origin is. It helps identify trade-offs. A non Power 4 school may be able to recruit higher rated players from certain positions. Depending on the factors, they may have to settle for lower stars.
